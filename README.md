# Amazon Sales Analytics

## 📊 Project Overview
SQL + Power BI Sales Dashboard using Amazon Sales Dataset from Kaggle.

## 🛠️ Technologies
- SQL Server 2022
- Power BI Desktop
- T-SQL (Views, Window Functions)

## 📁 Project Structure
- `SQL/01_analysis_queries.sql` — Analysis queries and Views for Power BI

## 📈 Dashboard Pages
1. **Category Analysis** — 1465 products, Avg Rating 4.09, Avg Price $1.53K
2. **Top Products** — Top 10 rated products, Price vs Rating scatter
3. **Discount Analysis** — Top 10 discounted products, Avg discount 47%

## 🔍 Key Insights
- Total Products: 1465
- Top Category: Electronics (35.9%)
- Average Rating: 4.09
- Average Discount: 47.75%
- Top Rated: Fire-Boltt
